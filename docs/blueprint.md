## Core Features:

- Project Gallery: Showcase creative work samples with detailed descriptions and visuals.
- 'About Me' Section: Dedicated section detailing professional background, skills, and experience.
- Contact Links: Integrated links to professional contact methods (LinkedIn: linkedin.com/in/sajansaji, GitHub: github.com/sajansaji, email: sajansaji2002@gmail.com).
- Resume Download: Downloadable version of resume (presumably in PDF format).
- Skill extraction: An AI-powered tool which takes the content of your resume as input and extract the core skills to showcase. The skill sections may include: Languages, AI/ML, Embedded/Linux, Tools, UI/UX

## Style Guidelines:

- Primary color: Gold (#D4AF37) to represent sophistication and creativity.
- Background color: Off-white (#FAF9F6) for a minimalist and clean backdrop.
- Accent color: Dark Gray (#333333) to provide contrast.
- Headline font: 'Playfair' serif with high-contrast, and a high-end feel.
- Body font: 'PT Sans' sans-serif to use with Playfair, for a humanist, modern feel.
- Use simple, elegant icons for navigation and key sections.
- Maintain a clean, grid-based layout for easy navigation.
